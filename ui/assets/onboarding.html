Welcome to SearchANDGraph

What this app does

This application runs targeted web and document searches and builds a knowledge graph from discovered content.

  • Run a scan with a query
  • View a generated interactive graph
  • Open documents linked as evidence

Starting a scan

Enter a concise query (example: Nvidia) and optionally a Start URL to focus crawling on a site.

If you don't provide a Start URL the web search may be broad and external network restrictions can block fetching resources.

Dealing with blocked content

If the graph fails to render or the scan seems to run but no HTML output appears, common causes are:

  • Your network or browser environment blocks remote resources (CDN or external scripts).
  • Search results returned external-only resources that were blocked from being embedded.

Recommendations: provide a Start URL, try again using a proxy / VPN, or re-run the scan with different options.

Options & Next steps

Open File → Options… to change the base folder, preferred sources, or enable/disable web search.

When a scan finishes successfully, use the Scans dropdown or the Open output folder button to inspect results.

Shortcuts

  • Keyboard: ←/→ to move between steps.
  • Use Start URL to improve focused crawls.
  • Prefer smaller max pages for quick tests.
