No HTML graph was found

The scan completed but no HTML graph was generated, or the embedded resources required to render it were blocked by your environment.

Why this can happen

  • Network policies or a restrictive browser environment block external scripts or CDNs.
  • The scan produced references to remote resources that could not be loaded when viewing locally.
  • Anti-virus, proxy, or VPN behaviour interfered with some downloads.

Recommended steps

  1. Provide a Start URL to scope crawling to a specific site and reduce external dependencies.
  2. Try running the scan again using a different network or a proxy/VPN.
  3. Check the README for troubleshooting tips.
